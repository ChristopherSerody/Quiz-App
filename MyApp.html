What It Is

The game pulls a random question for you to answer

What It Does

The game accesses a random Wikipedia page and asks the user to identify the subject of the page based on a hint.

Levelling Up

As you answer more questions correctly, the app accesses increasingly rarely cited pages so the questions get harder.

How It Works

The game uses machine learning to find new questions based on the difficulty of the questions you have previously answered.
